### 🚀 Tentang Saya

Saya adalah seorang **Developer Multi-Paradigma, AI Native Engineer** yang berfokus pada pengembangan sistem backend tangguh, pengalaman web premium, dan aplikasi lintas-platform berbasis AI. Keahlian saya mencakup pembangunan marketplace skala enterprise dengan **Clean Architecture** hingga ekosistem berperforma tinggi yang ditenagai **Machine Learning & AI**.

- 🛠️ Sedang mengembangkan **[MediMart.adv](https://github.com/rifqyhazim22/medimart.adv)** (Clean MVC), **[aplikasi-mirror](https://github.com/rifqyhazim22/aplikasi-mirror)** (Next.js/TensorFlow), dan **[ai-image-gen-app](https://github.com/rifqyhazim22/ai-image-gen-app)** (Flutter/AI).
- 📐 Ahli dalam **Next.js App Router**, **Flutter/Dart**, dan **Capacitor** untuk Mobile Native & Desktop (Electron).
- ⚡ Berfokus pada **UX Berbasis AI**, **Computer Vision Real-time**, dan **DX Berpusat Pengguna**.
- 🔐 Terampil menggunakan **Supabase**, **TensorFlow.js**, **MediaPipe**, **OpenAI API**, dan **ReactFlow**.
