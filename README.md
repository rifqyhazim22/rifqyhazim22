### 🏗️ Technical Architecture (The Reality)

The foundation of my work (as seen in `medimart.adv`) follows a strict **MVC & Clean Logic** structure:

```mermaid
graph TD
    User((User)) -- HTTP/WebSockets --> Express[Express Server]
    Express -- Middleware --> Auth[Auth / Session]
    Auth -- Controller --> Business[Business Logic]
    Business -- Sequelize ORM --> Postgres[(PostgreSQL Database)]
    Business -- Sharp API --> Images[Optimized Image Storage]
    Express -- EJS Engine --> UI[Responsive Frontend]
```
